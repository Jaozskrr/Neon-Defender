# NEON DEFENDER 🚀

Um jogo de arcade estilo Atari com alma moderna, desenvolvido para navegadores.

## 🕹️ Mecânicas
1. **Movimentação com Inércia**: Física de aceleração e frenagem.
2. **Mira por Mouse**: Os disparos seguem o cursor.
3. **IA de Perseguição**: Inimigos inteligentes que caçam o jogador.
4. **Power-ups**: Escudos temporários.
5. **Dificuldade Progressiva**: O desafio aumenta conforme sua pontuação.
6. **Highscore**: Recorde salvo localmente.
7. **Efeitos Visuais**: Explosões de partículas e estética CRT.

## 🎮 Controles
- **WASD / Setas**: Mover a nave
- **Mouse**: Mirar
- **Espaço**: Disparar / Iniciar jogo

## 🚀 Como Executar
Basta abrir o arquivo `index.html` em qualquer navegador moderno ou acessar o link do GitHub Pages (quando configurado).
